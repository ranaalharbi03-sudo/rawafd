# روافد الأنهار - Rawafid Al-Anhar

موقع متجر مياه يخدم المنازل والمدارس والمساجد في المملكة العربية السعودية.

## المميزات
- تشكيلة واسعة من المياه بجميع الأحجام والعلامات التجارية
- قوارير مياه كبيرة للمنازل والمكاتب
- باقات خاصة للمساجد والمدارس
- خدمة توصيل سريعة
- خيار التبرع (طوال الأسبوع / يوم الجمعة)

## التقنيات المستخدمة
- HTML5
- CSS3
- JavaScript (Vanilla)
- Google Fonts (Tajawal)
